# [팀 프로젝트] 항공권 예약 사이트

> **아시아나** 항공을 모티브로 사이트를 작성하였습니다. ~~!~~

> Spring MVC 패턴을 사용하여 만들게 되었습니다.

> 2019-11-01 ~ 2019-11-28

# 개발환경
* spring
* java
* javascript
* jquery
* jstl
* mysql
* mybatis
* css
* bootstrap
* ajax
* oracle
* tomcat

# 주요기능
* 로그인, ID찾기(가입 이메일 전송)
* 항공권 예약
* 회원가입
* 항공사 관리 페이지(공지사항 관리 및 상품관리, 항공기 등록,항공권 등록, 기타 등등)
* 게시판
* 이용 후기
* BEST 상품 확인가능
* 운항노선도
* 좌석 선택 및 기내식 선택
